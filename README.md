# Korbit Friend Referral (98145D)

A Korean landing page for sharing the Korbit friend referral code `98145D`.

## Deploy

- GitHub repository: `korbit-friend-referral`
- Vercel project: `korbit-friend-referral`
- Custom domain: `korbit.creamcode.xyz`

## DNS for Vercel

Add a CNAME record:

```text
Type: CNAME
Host: korbit
Value: cname.vercel-dns.com
```

Then add `korbit.creamcode.xyz` in Vercel Project Settings > Domains.
